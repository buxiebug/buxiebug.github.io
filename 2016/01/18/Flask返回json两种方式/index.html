2.使用response + json.dumps

打开浏览器访问：http://127.0.0.1:5000/json2/

两者都可以返回json数据，使用第二种可以更方便的自定义返回json的细节．例如，当json值中有中文时

jsonify默认ensure_ascii=True，导致非ascii字符被转码为unicode类型，导致了返回的数据是unicode编码的字符，可读性不高．

If ensure_ascii is false, all non-ASCII characters are not escaped, and
the return value may be a unicode instance. See dump for details.
