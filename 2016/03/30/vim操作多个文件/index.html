vim操作多个文件

2016-03-30
Linux

打开多个文件:

  • vim还没启动:
    • vim file1 file2
  • vim已经启动:
    • :open file

多个文件间切换：

  • ctrl+6 下一个文件
  • :bn 下一个文件
  • :bp 上一个文件

同时显示多个文件:

  • 水平显示:
    • :sp file
    • :split file
  • 垂直显示:
    • :vsp file
    • :vsplit file

多个窗格之间切换

  • ctrl+w+方向键 切换到上／下／左／右的窗格
  • ctrl+w+h/j/k/l 同上
  • ctrl+ww 依此向后切换到下一个窗格中
