python模拟微博登陆

2016-04-06
python
微博登陆
文章目录
  1. 1. 预登陆
  2. 2. 生成提交表单
  3. 3. 登陆并保存cookie

通过网上各种查阅资料以及自己的亲身实践,终于实现了使用python对新浪微博的模拟登陆,模拟登陆新浪微博主要分为三个部分:

  • 预登陆 用来获取登陆所需信息.这些信息用来加密传输数据,并提交给服务器做验证.
  • 生成提交表单
    • 生成用户名
    • 生成密码
    • 生成其他必须字段
  • 登陆并保存cookie
    • 请求登陆url并提交表单
    • 登陆成功后需要打开跳转页面完成登陆
    • 保存cookie,以便之后使用

预登陆

预登陆请求的url地址为http://login.sina.com.cn/sso/prelogin.php?entry=weibo&callback=sinaSSOController.preloginCallBack&su=ZW5nbGFuZHNldSU0MDE2My5jb20%3D&rsakt=mod&checkpin=1&cli,返回的数据是一个包含了json数据的html页面,通过正则表达式可以截取到json数据.

生成提交表单

既然是模拟登陆,用户名和密码就是最重要的两项

  • 用户名使用base64编码
  • 密码编码较为复杂:
    • 通过预登陆得到的pubkey的十进制(n)与65537(e)生成rsa公钥
    • 将预登陆得到的servertime,nonce和password拼接成字符串
    • 将第二步得到的字符串使用第一步生成的公钥加密
    • 将第三步得到的二进制加密数据转换为十六进制的数据,转换后的数据就是需要提交的密码
