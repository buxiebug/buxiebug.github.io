Fabric简化Python发布流程

2016-04-11
python
fabric

Python通用发布流程：

  • git checkout - b workspace
  • 编写逻辑
  • git add && git commit && git push
  • ssh登陆服务器
  • git pull
  • 启动脚本

在每次修改代码之后，都需要执行git提交，登陆服务器，更新代码，启动代码．登陆服务器，更新代码是一项重复枯燥无味的工作．

python使用Fabric可以轻松的在本地用Pythonic的方式完成ssh连接和执行shell命令．快速入门
