linux top

2016-04-19
Linux
Linux Command

top用来实时的查看系统的状态.

快捷键：

  • shift+p: 按照cpu使用率排序
  • shift+m: 按照内存使用率排序
  • shift+t: 按照cpu使用时间排序
  • c : 显示command详情
  • z 或 -b :　将状态为running的进程突出显示
  • k : kill pid 杀死进程
  • s : 设置刷新时间
